字体文件状态说明
================

当前字体配置已修改为使用本地字体文件，但字体文件尚未下载。

要完成字体配置，请执行以下操作之一：

1. 自动下载字体文件：
   npm run download-fonts

2. 手动下载并放置字体文件：
   - 从 Google Fonts 下载 Noto Sans JP 和 Noto Serif JP
   - 从 GitHub 下载 Geist Mono 字体
   - 将文件重命名并放置在此目录中

3. 临时使用系统字体：
   应用已配置了字体回退，即使没有字体文件也能正常运行

详细说明请查看 README.md 文件。
